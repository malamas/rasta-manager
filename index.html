    Rastaman is a student team project to develop a Radio Station Management application in Java.

    The application provides a (Swing) GUI to enable users to enter/modify/delete artist/album/song/playlist etc. info, and uses JavaDB (Apache Derby) for persistent storage through JPA.

    Το project Rastaman είναι ένα ομαδικό έργο με σκοπό την ανάπτυξη εφαρμογής διαχείρισης ραδιοφωνικού σταθμού σε Java.

    Η εφαρμογή παρέχει γραφικό περιβάλλον (σε Swing) που επιτρέπει στο χρήστη να εισάγει/μεταβάλλει/διαγράψει δεδομένα καλλιτεχνών/άλμπουμ/τραγουδιών/λιστών αναπαραγωγής κλπ., και χρησιμοποιεί JavaDB (Apache Derby) για μόνιμη αποθήκευση διαμέσου JPA.
